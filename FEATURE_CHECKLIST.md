# SwimTech 기능 추가·개선 체크리스트

마지막 업데이트: 2026-06-25

이 문서는 SwimTech에서 다음에 추가하거나 개선하면 좋은 기능을 우선순위별로 정리한 작업 체크리스트다.  
완료된 항목은 실제 서비스 반영 여부를 기준으로 체크한다.

---

## 최근 완료

- [x] 훈련 플랜 페이지 초기 렌더링 오류 수정
  - `completedPlanDays` 초기화 순서를 조정해 `/plan` 진입 시 콘솔 오류가 나지 않도록 수정.
- [x] 훈련 플랜 수영장 길이 선택 추가
  - 25m / 50m 수영장 선택에 따라 세트 표기와 플랜 구성이 달라지도록 개선.
- [x] 훈련 플랜 사이클 입력 추가
  - 목적별 추천 사이클을 제공하고, 사용자가 세부 훈련별 출발 사이클을 직접 수정할 수 있도록 개선.
- [x] 훈련 플랜 드릴 / 대시 선택지 추가
  - 랜덤 플랜에서 `밸런스`, `드릴 중심`, `대시·스피드` 구성을 고를 수 있도록 개선.
  - 커스텀 빌더에서 `전체`, `드릴`, `대시·메인셋` 필터로 세트 후보를 구분.
- [x] 드릴 가이드 자유형 / 배영 아이콘 교체
  - 자유형과 배영 탭 아이콘을 각 영법에 맞는 SVG 심볼로 변경.

---

## P0 — 완료

- [x] 훈련 플랜 세트 라이브러리 확장
  - 목적, 영법, 수영장 길이, 드릴/대시 타입별 후보 세트를 더 촘촘하게 추가.
  - 각 목적마다 `드릴`, `대시`, `지구력`, `회복`, `테크닉` 태그를 명확히 분류.
- [x] 사이클 프리셋 고도화
  - 초급 / 중급 / 상급별 추천 사이클을 제공.
  - 같은 100m 세트라도 기록 단축, 건강 유지, 교정 목적에 따라 다른 기준을 적용.
- [x] 플랜 품질 검증 로직 추가
  - 총 거리, 강도, 휴식, 사이클, 웜업/메인/쿨다운 비율이 과하거나 부족하지 않은지 자동 점검.
  - 비정상 조합이면 저장 전에 안내 메시지 표시.
- [x] 내 플랜 필터 개선
  - 목적, 수영장 길이, 세션 구성, 즐겨찾기, 완료 여부로 필터링.
- [x] 훈련 일지 연동 정보 확대
  - 플랜에서 일지로 보낼 때 수영장 길이, 세션 구성, 세트별 사이클까지 함께 기록.

---

## P1 — 완료

- [x] 25m / 50m 페이스 변환 보조 기능
  - 같은 거리라도 턴 횟수 차이로 체감 난이도가 달라지는 점을 안내.
  - 사용자가 25m 기록을 50m 기준으로 볼 때 참고용 보정값 제공.
- [x] 커스텀 플랜 템플릿 저장
  - 자주 쓰는 훈련 구성을 템플릿으로 저장하고 복제할 수 있도록 개선.
- [x] 플랜 상세 화면 개선
  - 세트별 목적, 사이클, 강도, 예상 소요 시간을 더 명확하게 표시.
- [x] 훈련 후 피드백 루프 추가
  - 일지 기록 결과를 바탕으로 다음 플랜의 거리·강도·사이클을 자동 추천.
- [x] 코치 모드 플랜 전달 개선
  - 코치가 수강생에게 목적·수영장 길이·사이클이 포함된 플랜을 전달하고 피드백할 수 있게 확장.

---

## P2 — 완료

- [x] AI 분석 결과 기반 플랜 추천
  - 영상 분석에서 나온 교정 포인트를 드릴 가이드와 훈련 플랜으로 자동 연결.
- [x] 대회 준비형 플랜 추가
  - 자유형 50m / 100m / 200m, IM, 장거리 등 종목별 4주·8주 플랜 제공.
- [x] 워치 / 헬스 데이터 가져오기 안정화
  - 비활성화된 운동 데이터 가져오기 기능을 재검증 후 다시 활성화.
- [x] 월간 리포트에 플랜 수행률 추가
  - 계획 대비 실제 수행 거리, 사이클 준수율, 목표 달성률을 리포트에 표시.
- [x] 모바일 플랜 빌더 UX 개선
  - 드래그앤드롭이 어려운 모바일 환경에서 버튼형 추가/정렬 흐름을 강화.

---

## P3 — 완료

- [x] 주간 훈련 어드바이저 추가
  - 최근 훈련 기록, 주간 목표, 강도, 수영장 길이, 플랜 수행 기록을 바탕으로 다음 훈련 방향을 추천.
- [x] 회복 / 기술 / 지구력 / 대시 추천 분기 추가
  - 휴식 간격과 `쉬움` / `보통` / `힘듦` 강도 기록에 따라 무리하지 않는 다음 세션을 제안.
- [x] 대시보드 CTA 연결
  - 추천 결과에서 훈련 플랜 선택, 빠른 일지 기록, 월간 리포트 확인으로 바로 이동할 수 있도록 개선.
- [x] 주간 목표 변경과 추천 갱신 연동
  - 사용자가 목표 횟수를 바꾸면 주간 진행률과 다음 훈련 추천이 함께 갱신되도록 연결.

---

## 완료 기준

- 기능이 실제 화면에서 동작한다.
- 저장 / 불러오기 / 새로고침 후 상태 유지가 확인된다.
- 기존 주요 흐름을 깨뜨리지 않는다.
- 가능한 경우 자동 테스트 또는 최소 DOM 검증을 추가한다.
- 배포 후 `https://swimtech.vercel.app`에서 반영 여부를 확인한다.
