👤 이용자 권리

이용자는 언제든지 아래 권리를 행사할 수 있습니다.

  • 열람: 본인의 개인정보 확인
  • 정정: 잘못된 정보 수정 요청
  • 삭제: 개인정보 및 분석 결과 삭제 요청
  • 처리정지: 개인정보 처리 정지 요청

요청은 서비스 내 설정 메뉴 또는 이메일(privacy@swimtech.app)로 접수하시면 5영업일 내 처리합니다.

🔒 보안 조치

  • 비밀번호 단방향 암호화(bcrypt) 저장
  • 서버 간 통신 HTTPS(TLS 1.3) 적용
  • DB 접근 권한 최소화 및 접근 로그 기록
  • 업로드 영상 서버 내 격리 저장, 분석 후 자동 정리
  • 세션 토큰 만료 시간 적용 (JWT, 7일)

🍪 쿠키 사용

서비스는 로그인 상태 유지를 위해 쿠키(swimtech_token)를 사용합니다. 브라우저 설정에서 쿠키를 비활성화하면 로그인이 유지되지 않을 수 있습니다. 개인 식별 목적의 광고 쿠키는 사용하지 않습니다.
