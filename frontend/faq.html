❓ 자주 묻는 질문

궁금한 점을 검색하거나 카테고리별로 찾아보세요

🔍
수영장 옆에서 측면으로 촬영하면 팔꿈치 각도와 자세 분석에 가장 유리해요.
전신이 나오도록 충분한 거리를 두고 촬영해주세요.
수중 카메라로 촬영하면 발차기 분석에 더 유리합니다.
30초~3분 사이가 적당해요. 너무 짧으면 발차기 횟수 측정이 어렵고, 너무 길면 분석 시간이 오래 걸려요. 10~20 스트로크가 포함된 영상을 권장합니다.
역광이나 어두운 환경에서는 AI가 신체 랜드마크를 감지하기 어려워 분석 정확도가 낮아질 수 있어요. 밝은 환경에서 촬영을 권장합니다.
mp4, mov, avi, mkv, webm 형식을 지원합니다.
파일 크기는 최대 500MB (모바일은 100MB 권장)입니다.
SwimTech는 Google MediaPipe 기반으로 33개 신체 랜드마크를 추적하고, 스포츠 역학 논문(PMC4000476, PMC4738963 등)의 기준값을 적용해요. 촬영 환경과 영상 품질에 따라 정확도가 달라질 수 있습니다.
영상 길이에 따라 30초~3분 정도 소요됩니다. 분석 중에는 실시간으로 진행 상황을 확인할 수 있어요.
자유형, 배영, 평영, 접영, 스타트, 플립턴, 터치턴 7가지 영법을 지원합니다. 영상 업로드 전 영법을 선택하면 더 정확한 분석 기준이 적용됩니다.
팔꿈치 각도, 좌우 대칭, 발차기 빈도, 머리/시선 자세 4가지 항목을 선택한 수영 목적에 따른 가중치로 합산해 0~100점으로 표시합니다.
로그인 페이지에서 Google 또는 카카오 버튼을 클릭하면 됩니다. 처음 소셜 로그인 시 닉네임을 설정하는 페이지로 이동합니다.
분석 완료 후 바로 확인할 수 있고, 로그인 후 대시보드에서 이전 분석 결과를 언제든지 다시 볼 수 있어요.
분석 완료 후 "Google Sheets에 저장" 버튼을 클릭하면 처음 한 번만 Google 계정 인증 후 자동으로 저장됩니다.
메인 메뉴에서 "수영장 찾기"를 클릭하면 현재 위치 기준으로 주변 수영장을 지도에서 확인할 수 있어요.
분석 완료 후 "결과 공유하기" 버튼을 클릭하면 링크가 복사됩니다. 공유 링크는 로그인 없이 누구나 볼 수 있어요.
네! SwimTech는 PWA를 지원해 모바일 홈 화면에 설치할 수 있어요. 갤러리에서 영상을 직접 업로드하거나 카메라로 바로 촬영도 가능합니다.
검색 결과가 없습니다. 다른 키워드로 검색해보세요.
